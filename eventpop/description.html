What is JavaScript Bangkok 1.0.0?

JavaScript Bangkok 1.0.0 is an independently organized, single-day, single-track conference about everything JavaScript where both local and international JavaScript communities unite and learn from each other.

The event will take place on Saturday February 8th, 2020 at KBank Siam Pic-Ganesha Theater, located at Siam Square One, situated right at the heart of Bangkok, connected to the Siam BTS central station.
